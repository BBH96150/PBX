What this does

Webhooks POST a signed JSON payload to your URL when events happen on this workspace. Each request includes an X-Webhook-Signature: sha256=… header — an HMAC-SHA256 of the raw body using the endpoint's secret — so you can verify it's genuinely from us. Delivery is best-effort; we don't retry.
