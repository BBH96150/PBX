Call park puts an in-progress call on hold in a numbered "orbit" slot so any phone can pick it back up.

  • Feature code — blind-transfer a live call to this code (e.g. *68) to park it. FreeSWITCH auto-assigns the next free slot and announces it.
  • Slot range — the orbit numbers parked calls land in (e.g. 700–779). Dial the announced slot from any phone to retrieve the call.
