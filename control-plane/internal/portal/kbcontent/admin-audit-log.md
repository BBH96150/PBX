# Audit Log

The audit log records security- and administration-relevant events in your workspace — sign-ins, invites, configuration changes, and more — so you can review who did what and when. Find it under **Admin ▾ → Audit log**.

## Review events

1. Go to **Admin ▾ → Audit log**.
2. Each entry shows the time, the actor (email), the event, the affected target, and the source IP / user agent.

Logged events include things like successful and failed sign-ins, blocked logins, invites, and bulk actions.

## Export for compliance

Download the audit log as a CSV from the audit page (the `audit.csv` export) for security review or record-keeping. It includes the timestamp, actor, event, target, IP, user agent, and event details.
