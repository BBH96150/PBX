# Set Up Voicemail and Voicemail-to-Email

Voicemail lets callers leave a message when an extension is unanswered, busy, or in Do Not Disturb. You can also have each new message emailed automatically.

## Turn voicemail on

1. Open the extension's detail page.
2. On the **Features** card, check **Voicemail enabled** and save.

## Create the voicemail box

Once voicemail is enabled, a **Create voicemail box** card appears on the extension page:

1. Set a **PIN** (4–12 digits) the user enters to check messages by phone.
2. Optionally enter an **Email** address — every new message will be emailed there (voicemail-to-email).
3. Click **Create**.

## Listen to and manage messages

Messages appear in the **Voicemail box → Inbox** on the extension page (and in the owner's [self-service area](self-service-portal.md)). For each message you can:

- Play it right in the browser.
- Download the recording.
- Delete it.

New messages are marked **new**.

## Use voicemail as an after-hours destination

A voicemail box is a valid routing destination. You can send a phone number, an IVR option, or after-hours calls straight to voicemail — pick the voicemail box from the destination dropdown. See [Route inbound phone numbers (DIDs)](numbers-route-dids.md) and [Business hours](numbers-business-hours.md).
