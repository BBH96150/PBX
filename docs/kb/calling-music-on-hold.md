# Music on Hold

Music on hold is what callers hear while they wait — on hold, or while queued. You can set a default source for your whole workspace, and override it per call queue.

## Set the workspace default

1. Open your workspace **Overview** page.
2. Find the **Music on hold** setting.
3. Enter the source (a stream or sound source path) and save.

## Override it for a specific queue

1. Open the call queue's detail page.
2. Expand **Edit settings**.
3. Set the **Music on hold** value for that queue (for example `local_stream://moh`).
4. Click **Save settings**.

A queue's own music-on-hold setting takes precedence over the workspace default for callers waiting in that queue.
